# Expense Logger (MongoDB Project)

A simple Expense Logger application built using Flask and MongoDB to understand NoSQL databases and CRUD operations.

## Features
- Add daily expenses
- View all expenses
- MongoDB database storage

## Tech Stack
- Python
- Flask
- MongoDB
- HTML & CSS

## MongoDB Concepts Used
- Database & Collections
- Insert & Find operations
- Sorting data

## How to Run
1. Install requirements
   pip install -r requirements.txt

2. Start MongoDB
   mongod

3. Run the app
   python app.py

4. Open browser
   http://127.0.0.1:5000/
